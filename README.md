# Kitchen Companion v0.4

Responsive recipe and kitchen app for phone and desktop.

## New in v0.4

- Mobile-safe Favorite and Edit action row
- Existing-category dropdown with custom category creation
- Personal edits remain overlays; imported originals are preserved
- Modified personal recipes can open the original
- Automatic cooking-time detection in instruction steps
- Multiple timers with pause, resume, add one minute, and cancel
- Timer ranges offer an early check time or the longer time
- Existing local recipes, notes, favorites, modules, and settings are migrated in place
- Corrected two vegetable dishes that were categorized as sauces
