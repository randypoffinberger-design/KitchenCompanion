# Kitchen Companion v0.10.2

Profile system completion release.

- Adds profile import for `.kcprofile` exports.
- Detects matching profile IDs and offers replace or separate-copy import.
- Stores personal recipes directly as profile data while preserving the My Recipes module presentation.
- Automatically migrates older `personalModule` profile data.
- Improves mobile profile-card layout and action hierarchy.
- Adds last-used information and a local storage/backup summary.
- Makes Shared Kitchen profiles visually distinct.
- Keeps old v0.10.0 and v0.10.1 exports compatible.
