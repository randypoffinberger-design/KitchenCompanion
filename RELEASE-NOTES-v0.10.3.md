# Kitchen Companion v0.10.3

## Active profile and quick switching

- Shows the active profile beside the Kitchen Companion name in the main header.
- Tapping the profile name opens a compact quick-switch menu.
- The quick switcher lists every local profile and marks the current one.
- Includes a Manage profiles link for opening the full profile manager.
- Profile menus close when tapping outside them or pressing Escape.
- Opening one More menu closes any other open profile menu.
- Updated app cache and package metadata.
