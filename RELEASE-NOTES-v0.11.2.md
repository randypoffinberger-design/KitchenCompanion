# Kitchen Companion v0.11.2

## Shopping List 2.0: compact rows and bulk actions

- Shopping rows now show only the item name and store by default.
- Tap an item name to reveal quantity, recipe source, reason, Edit, and Remove.
- Added a sticky Select bar that remains visible while scrolling.
- Select multiple visible items, cancel safely, select all visible, move them to one store, or delete them together.
- Bulk delete asks for confirmation.
- Bulk move and delete include a seven-second Undo action.
- Selection mode keeps shopping checkboxes separate from bulk-selection controls.
