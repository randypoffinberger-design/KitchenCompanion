# Kitchen Companion v0.11.5

## Data safeguards
- Added up to five rolling local safety checkpoints.
- Added manual checkpoint creation and restore controls.
- A fresh checkpoint is created before restoring an older checkpoint.
- Added storage schema, migration, profile, module, and backup diagnostics.
- Added a built-in smoke-test panel for core data structures and browser storage support.
- Added a development journal and formal release checklist.
- Corrected all cache, manifest, script, and service-worker versions to 0.11.5.

Safety checkpoints remain local to the device. Full exported backups are still recommended before major upgrades or device changes.
