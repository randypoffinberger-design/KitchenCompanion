# Kitchen Companion v0.11.3

## Shopping list density cleanup

- Replaced the oversized bulk-action panel with a single sticky compact row.
- Normal mode now shows only **Select items**.
- Selection mode shows **Cancel**, selected count, **All/None**, **Move**, and **Delete** in one row.
- Bulk Move now opens a compact store picker only when needed.
- Replaced each full-width per-item store dropdown with a small tappable store pill.
- Ingredient rows now remain one line by default, with a separate details chevron.
- Preserved bulk delete confirmation and seven-second Undo.
