# Kitchen Companion v0.11.4

- Fixed the normal shopping toolbar so hidden selection controls no longer consume horizontal space.
- Added a two-row mobile selection toolbar for Cancel, count, All/None, Move, and Delete.
- Added “New store…” to the bulk and single-item Move dialog.
- New stores can be created and immediately assigned with Add & Move.
- Duplicate store names are blocked case-insensitively.
- Newly created stores are saved to Manage Stores and all future store pickers.
